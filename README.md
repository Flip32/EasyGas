# easygas


# Dependências necessárias

###  Ter o IONIC
###  Ter o NPM
```
$ npm install -g ionic cordova
```


## Para fazer Funcionar



```bash
cd /path/to/project
npm install

```

## Para rodar a aplicação local

A aplicação, por default, fica disponível em `localhost:8100`

```bash

ionic serve

```
